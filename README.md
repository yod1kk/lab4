# Лабораторна робота №4: Створення графів та алгоритми їх обходу (Частина 2)

Цей репозиторій містить реалізацію розширених алгоритмів на графах мовою Java.
##  Реалізовані алгоритми
В одному класі `Lab4Graphs.java` реалізовано:
1. **Виявлення циклів у неорієнтованому графі** 
2. **Навігація картою міста (Пошук найкоротшого шляху та відстаней)**
3. **Пошук альтернативних шляхів**
4. **Перевірка графу на двочастковість ** 
##  Досліджувані графи
Усі графи представлені в пам'яті як **списки суміжності** (через двовимірні масиви `int[][]`), що забезпечує оптимальне використання оперативної пам'яті ($O(V + E)$).
